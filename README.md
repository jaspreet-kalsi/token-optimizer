Token Optimizer

Optimize text and code before sending to AI — reduce token usage without losing meaning.

Features

3 optimization modes — Text, Code, and Aggressive (both combined)
In-place injection — Optimize button appears inside the AI chat input, no copy-paste needed
Live savings stats — see token count, % saved, and context window usage in real time
5 model targets — Claude 200k, GPT-4 128k, Gemini 2M, Mixtral 32k, Llama 8k
Works on ChatGPT, Claude, Gemini, Mistral, Poe, Perplexity, Copilot
